== Description ==

An AJAX styled form that allows you to receive guest posts. The user is allowed to user hyperlinks in both the Body and Author's Bio. You can control from the backend if you want the hyperlinks to be 'Do Follow' or not plus a few other features. Simply place the short-code within a page or post and you are ready to go.

== Installation ==

1. Upload `BestGuestPoster` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new user (Not necessary but recommended).
4. Create a new category to place new submissions in (Not necessary but recommended)
5. Make some final changes in the settings (Located in Wordpress Settings)
6. Place shortcode [ByWDclub.com] on a page or post.
